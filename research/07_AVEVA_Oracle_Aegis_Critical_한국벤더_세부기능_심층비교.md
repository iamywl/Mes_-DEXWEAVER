## Part A. 글로벌 MES 4사 세부 기능 심층 비교

---

### 1. 생산 스케줄링 / APS (Advanced Planning & Scheduling)

#### 1.1 AVEVA MES

| 항목 | 상세 내용 |
|------|----------|
| **유한 용량 스케줄링** | AVEVA 자체 APS가 아닌 **PlanetTogether APS**와의 파트너십으로 제공. PlanetTogether의 유한 용량 알고리즘이 자원 용량, 원부자재 가용성, 설비별 스케줄링 규칙을 동시에 고려하여 스케줄 생성. What-If 시뮬레이션으로 여러 시나리오 비교 가능 |
| **AI/ML 기반 최적화** | PlanetTogether이 ML 기반 수요 예측 및 공급망 중단 예측 기능 제공. AI 기반 제약 조건 스케줄링으로 셋업 최소화 및 납기 준수율 최적화. AVEVA 자체적으로는 AVEVA Intelligence 플랫폼을 통한 분석 제공 |
| **실시간 리스케줄링** | PlanetTogether가 MES 생산 피드, 자원 상태, 수요 변동을 실시간으로 반영하여 스케줄을 동적 갱신. 단, AVEVA MES와 PlanetTogether 간 연동은 별도 통합 작업 필요 |
| **차별점** | 모델 기반 MES 철학에 따라 공정 모델을 정의하면 스케줄링 규칙도 모델에 포함. 연속공정/배치 프로세스 스케줄링에 강점. PI System(현 AVEVA Data Hub)의 히스토리컬 데이터를 스케줄링 최적화에 활용 가능 |

#### 1.2 Oracle Fusion Cloud Manufacturing

| 항목 | 상세 내용 |
|------|----------|
| **유한 용량 스케줄링** | Oracle Supply Chain Planning Cloud 내 **Production Scheduling** 모듈이 유한 용량 스케줄링 제공. 실시간 자원 가용성과 작업 오더를 기반으로 자재, 용량, 캘린더 제약 조건을 고려한 실행 가능 스케줄 생성. 공장 처리량 증대 및 WIP 재고 감소에 초점 |
| **AI/ML 기반 최적화** | **2026년 26A 릴리스**에서 AI 에이전트 대폭 확대: Planning Cycle Agent(계획 태스크 자동화), Work Order Scheduling Agent(작업 오더 스케줄링 최적화), Inventory Tasking Agent(창고 작업 배분). 추가 비용 없이 Oracle Fusion에 내장. Oracle Fusion AI Data Platform 기반 예측 분석 |
| **실시간 리스케줄링** | Smart Operations가 실시간 운영 데이터와 비즈니스 데이터를 결합하여 권고사항 제공. 생산 감독관 워크벤치에서 GenAI 기반 교대 보고와 실시간 인사이트로 작업 오더 관리 |
| **차별점** | **SaaS 전용**이므로 분기별 자동 업데이트로 최신 AI 기능을 추가 비용 없이 획득. ERP/SCM과 네이티브 통합되어 수요 계획-생산 계획-실행이 단일 플랫폼에서 완결. 이산/프로세스 혼합 모드(Mixed-Mode) 제조를 동일 공장에서 지원하는 것이 고유 강점 |

#### 1.3 Aegis FactoryLogix

| 항목 | 상세 내용 |
|------|----------|
| **유한 용량 스케줄링** | **Adaptive Planning** 모듈로 제공. 라이브 규칙 기반 엔진이 작업, 자재, 기계, 제품 구조를 실시간 모델링. 드래그 앤 드롭 인터랙티브 스케줄 보드에서 작업 배치 시 자동으로 소요 시간 계산, 자재 예약, 설비 가용성 충돌 감지. 유한 용량을 명시적으로 지원하되, 전통적 APS 엔진과는 접근법이 다름 |
| **AI/ML 기반 최적화** | 전통적 AI/ML 기반 자동 최적화 엔진은 내장되어 있지 않음. 대신 IIoT 데이터 기반의 실시간 팩토리 인식(Factory Awareness)으로 자재 부족, 설비 가용성 문제를 사전 경고. Simio 시뮬레이션 도구와의 파트너십으로 디지털 트윈 기반 스케줄링 시뮬레이션 가능 |
| **실시간 리스케줄링** | 가장 큰 차별점: **즉각적 적응형 리스케줄링**. 가상 프로세스 라우팅이 실행 시점에 물리적으로 구현되므로, 긴급 라인 상황, 공장 간 제품 이관, 단일 개체 흐름(Single-Piece Flow), MRO/RMA 처리가 즉시 적응. 스케줄 변경이 실시간 IIoT 기계 통신 및 자재 소비와 연동 |
| **차별점** | 전자 조립(SMT/PCBA) 환경에 최적화된 스케줄링. 기계별 통신(IIoT 어댑터)과 자재 소비 상황을 실시간으로 반영하여 공급망 중단이나 설비 경합에 의한 다운타임 회피. 별도 APS 라이선스 불필요(MES에 내장) |

#### 1.4 Critical Manufacturing MES

| 항목 | 상세 내용 |
|------|----------|
| **유한 용량 스케줄링** | 내장 **APS 모듈**이 다중 기준 공장 레벨 최적화 수행. 설비 활용률, 납기 준수, 셋업 최소화, 사이클 타임 등 복수 KPI를 동시 최적화. 작업 스케줄, 자원 가용성, 인원 자격 인증, 자재 의존성, 납기일 등 다양한 제약 조건 준수. 각 설비별 처리 순서 자동 생성 |
| **AI/ML 기반 최적화** | 고급 분석 및 휴리스틱 알고리즘 기반 최적화. 별도 AI/ML 엔진이라기보다 고급 알고리즘과 제약 만족 기법(Constraint Satisfaction)으로 스케줄 최적화. IoT 데이터 플랫폼(Connect IoT)에서 수집된 실시간 설비 데이터를 활용 |
| **실시간 리스케줄링** | 실시간 설비 상태 및 WIP 위치 데이터를 반영한 동적 스케줄 조정. 반도체 Fab 환경의 복잡한 재진입 공정(Re-entrant Flow)과 다단계 패키징 지원 |
| **차별점** | **반도체/전자 특화 스케줄링**이 핵심 차별점. 웨이퍼 맵핑, 기판 맵핑, 복잡한 제품 흐름, 다단계 패키징 등 반도체 고유 요구사항 지원. APS가 MES 내 네이티브 모듈로 통합되어 별도 제품 구매 불필요. 로우코드로 스케줄링 규칙 커스터마이징 가능 |

### 2. 작업지시 관리 (Work Order / Work Instruction Management)

#### 2.1 AVEVA MES

| 항목 | 상세 내용 |
|------|----------|
| **전자 작업지시서** | **AVEVA Work Tasks** 모듈이 디지털 워크플로 관리 솔루션으로 통합. 작업자에게 단계별 가이드 제공. 디지털 워크플로를 표준화하여 모든 사이트에서 일관된 작업 수행 보장 |
| **작업 순서 강제/가이드** | 모델 기반 접근: 공정 모델에 작업 순서를 정의하면 시스템이 순서를 강제(Enforce). 바이패스 방지 및 필수 체크포인트 검증. ISA-88 기반 배치 순서 강제 및 "done-by"/"check-by" 보안으로 전자서명 첨부 |
| **작업자 인터페이스** | HTML5 기반 씬 클라이언트로 PC/모바일 기기에서 접근 가능. **오프라인 모드 지원**: 디바이스에 작업 태스크를 다운로드하여 네트워크 연결 없이도 작업 수행 및 데이터 입력 가능. 재연결 시 자동 동기화 |
| **차별점** | 오프라인 워크플로가 핵심 차별점. 네트워크 불안정한 플랜트 환경(광업, 원격지 공장)에서 유리. 모델 기반이므로 작업지시 템플릿을 한번 정의하면 다수 사이트에 복제 가능 |

#### 2.2 Oracle Fusion Cloud Manufacturing

| 항목 | 상세 내용 |
|------|----------|
| **전자 작업지시서** | Work Definition(작업 정의)에 기반한 전자 작업 오더 시스템. 공정별 오퍼레이션, 자원, 자재를 정의하고 작업 오더로 인스턴스화. 감사추적(Audit Trail)이 생성/수정/삭제에 대해 자동 기록 |
| **작업 순서 강제/가이드** | 작업 정의에서 오퍼레이션 순서 및 의존관계를 설정하면 실행 시 순서 강제. 프로세스 제조에서는 포뮬라-레시피-배치 동기화로 레시피 변경 시 영향 받는 배치를 AI 기반 What-If 시나리오로 자동 업데이트 |
| **작업자 인터페이스** | **모바일 산업용 핸드헬드** 기기 지원: 자재 투입/반환, 오퍼레이션 완료, 제품 산출 기록을 바코드 스캐닝으로 수행. 생산 보고 작업 영역(Production Reporting Work Area)에서 트랜잭션 수행. **생산 감독관 워크벤치**: GenAI 기반 교대 보고로 실시간 작업 오더 인사이트 제공 |
| **차별점** | ERP-MES 완전 통합으로 작업 오더가 ERP 자재/원가와 실시간 연동. GenAI 기반 교대 보고는 업계 최초 수준의 AI 활용. 모바일 핸드헬드 지원으로 현장 작업자 편의성 높음 |

#### 2.3 Aegis FactoryLogix

| 항목 | 상세 내용 |
|------|----------|
| **전자 작업지시서** | **가장 진보된 비주얼 작업지시서** 제공. BOM과 CAD(ECAD/MCAD) 데이터를 활용하여 자동으로 인터랙티브 비주얼 지시서 생성. 전자/기계 조립 모두에 대해 가장 상세한 수준까지 3D 뷰 제공. 작업자가 3D 모델을 회전/확대하여 상세 정보 확인 가능 |
| **작업 순서 강제/가이드** | 그래픽 흐름도(Flow Diagram)로 라우팅 설계. 논리 조건과 경로를 포함한 가상 프로세스가 실행 시점에 물리적으로 구현. 이 가상-물리 분리 구조가 긴급 라인 변경, 공장 간 이관, 단일 개체 흐름, MRO/RMA 자동 처리를 가능하게 함 |
| **작업자 인터페이스** | **FactoryOptix**: 증강현실(AR) 기반 작업 지시 솔루션. 동일 저작 시스템에서 작성한 비주얼을 AR 디바이스로 전송. 전자 조립의 각 부품 위치를 AR로 실시간 안내. 키오스크/태블릿 기반 터치스크린 인터페이스 |
| **차별점** | CAD 연동 자동 작업지시 생성 및 AR 지원이 최고 차별점. 커스터마이징 없이 2,200개 이상의 제조 사이트에서 동일 소프트웨어 사용(구성 가능성). 전자 조립 분야에서 가장 상세한 수준의 비주얼 가이드 제공 |

#### 2.4 Critical Manufacturing MES

| 항목 | 상세 내용 |
|------|----------|
| **전자 작업지시서** | ECAD 파일 로딩 및 시각화를 통한 작업지시서 생성 지원. BOM과 연계된 작업 단계별 지시사항 정의. 디지털 트윈과 연결된 작업지시로 실시간 상황 반영 |
| **작업 순서 강제/가이드** | 자재 라우트 실행을 강제(Enforce)하고 관련 처리 구성을 자동 적용. 풀 자재 추적성 및 계보(Genealogy)와 연결. 반도체 Fab의 복잡한 재진입 공정(같은 설비를 여러 번 거치는 플로)에 대한 순서 강제 지원 |
| **작업자 인터페이스** | 직관적인 웹 기반 UI(Angular/TypeScript). **증강현실(AR)** 내장으로 작업자에게 실시간 오버레이 정보 제공. 디지털 트윈 대시보드에서 운영 메트릭 실시간 확인. **로우코드 확장**: 비개발자도 워크플로와 화면을 커스터마이징 가능 |
| **차별점** | 로우코드 확장성이 핵심. UI/UX를 코딩 없이 변경 가능하여 공장별 요구에 빠르게 대응. AR 및 디지털 트윈이 MES에 네이티브 통합된 것은 경쟁사 대비 독보적 |

### 3. 품질 관리 (Quality Management)

#### 3.1 AVEVA MES

| 항목 | 상세 내용 |
