## 2. 솔루션별 상세 기능 특장점

### 2.1 Siemens Opcenter

**한 줄 요약**: 이산/프로세스/하이브리드 제조를 모두 아우르는 업계 최고 수준의 종합 MOM 플랫폼

| 특장점 | 상세 설명 |
|--------|---------|
| 포괄적 MOM 포트폴리오 | 생산 실행, APS, 품질, 연구개발, 성능 분석을 단일 브랜드로 제공 |
| PLM 네이티브 통합 | Teamcenter/Simcenter/Tecnomatix와 심리스 연계 → 설계-제조 연속성 확보 |
| Mendix 로우코드 | UI 개발 시간 50% 단축, 비개발자도 화면 구성 가능 |
| Opcenter X (SaaS) | 클라우드 SaaS 버전으로 중소기업 진입장벽 대폭 낮춤 |
| 산업별 특화 버전 | 반도체, 제약, 전자, 자동차 등 산업별 프리셋 템플릿 제공 |
| Insights Hub 연계 | 클라우드 기반 데이터 분석/AI 활용 |

**적합 대상**: 대기업, 멀티사이트 글로벌 운영, 이산+프로세스 복합 제조
**주의점**: 높은 초기 도입 비용, Siemens 에코시스템 종속 가능성

---

### 2.2 Rockwell Automation - Plex MES

**한 줄 요약**: 업계 유일의 클라우드 네이티브 싱글 인스턴스 SaaS MES

| 특장점 | 상세 설명 |
|--------|---------|
| 클라우드 네이티브 | 멀티테넌트 SaaS로 설계, 별도 인프라 없이 즉시 배포 |
| Edge-to-Cloud | 현장 엣지 데이터와 클라우드 분석의 하이브리드 구조 |
| 통합 플랫폼 | MES + ERP + QMS + APM이 단일 데이터 소스로 통합 |
| 모듈형 확장 | 필요한 모듈만 선택해서 점진적으로 확장 가능 |
| 실시간 단일 소스 | 모든 공장 데이터가 하나의 진실(Single Source of Truth) |

**적합 대상**: 클라우드 우선 전략 기업, 중견기업, 식음료/자동차/소비재
**주의점**: Rockwell 생태계 외부 연동 시 제약, ERP 호환성 확인 필요

---

### 2.3 SAP ME/MII/DMC

**한 줄 요약**: SAP ERP 생태계와의 최강 연계력, 단 차세대 DMC로의 전환 필수

| 특장점 | 상세 설명 |
|--------|---------|
| S/4HANA 네이티브 통합 | ERP-MES 간 실시간 데이터 교환, 별도 미들웨어 불필요 |
| SAP 생태계 시너지 | PLM, SCM, QM, PP 등 전체 SAP 모듈과 원활한 연계 |
| SAP DMC (차세대) | 클라우드 네이티브 디지털 제조 솔루션으로 AI/IoT 통합 강화 |
| 글로벌 검증 실적 | 대기업 중심 풍부한 도입 레퍼런스 |

**적합 대상**: SAP ERP 사용 대기업, 자동차/항공우주/화학 산업
**주의점**: SAP ME/MII는 **2030년 12월 EOL** → DMC 마이그레이션 계획 필수

---

### 2.4 Dassault DELMIA Apriso

**한 줄 요약**: Virtual Twin 기술로 "디지털 쌍둥이" 시뮬레이션을 MES에 통합

| 특장점 | 상세 설명 |
|--------|---------|
| Virtual Twin | 실제 공장의 가상 복제본으로 "What-if" 시나리오 시뮬레이션 |
| 3DEXPERIENCE 통합 | PLM(CATIA, ENOVIA)과의 긴밀한 설계-제조 연계 |
| 빠른 Go-Live | 로우코드 환경으로 90일 이내 초기 가동 가능 |
| 글로벌 표준화 | 멀티사이트 배포 시 공정 표준화에 강점 |

**적합 대상**: 항공우주/국방, 자동차, Dassault PLM 사용 기업
**주의점**: 높은 가격, 비-Dassault 환경 독립 운영 제한

---

### 2.5 Honeywell MXP

**한 줄 요약**: 제약/화학 등 프로세스 산업에 최적화된 통합 플랫폼

| 특장점 | 상세 설명 |
|--------|---------|
| 올인원 플랫폼 | MES + SCADA + Historian을 단일 플랫폼에 통합 |
| Experion DCS 통합 | 프로세스 제어-실행 시스템 간 심리스 연계 |
| 규정 준수 | 21 CFR Part 11, GxP 규정 준수 내장 |
| 노코드/로우코드 | 템플릿 라이브러리로 빠른 배포 |

**적합 대상**: 제약/생명과학, 화학, 석유/가스 (프로세스 산업)
**주의점**: 이산 제조 기능 한계, Honeywell DCS 외부 환경 통합 제약

---

### 2.6 GE Vernova Proficy

**한 줄 요약**: 2024년 Kubernetes 기반 완전 재구축으로 현대화된 하이브리드 MES

| 특장점 | 상세 설명 |
|--------|---------|
| 컨테이너 기반 | 2024년 Kubernetes 기반 완전 재구축, 현대적 마이크로서비스 |
| Unified Namespace | UNS 아키텍처로 실시간 데이터 통합 |
| Proficy Historian | 업계 최고 수준 데이터 히스토리안과 네이티브 통합 |
| 멀티 제조 유형 | 이산, 프로세스, 하이브리드 제조 모두 지원 |

**적합 대상**: 식음료, 제약, CPG, 화학 (하이브리드 제조)
**주의점**: 설정 복잡도 높음, 전문 인력 필요, 월 $5,800부터

---

### 2.7 AVEVA MES

**한 줄 요약**: 모델 기반 아키텍처로 멀티사이트 표준화에 탁월한 연속공정 MES

| 특장점 | 상세 설명 |
|--------|---------|
| 모델 기반 MES | 공정 모델을 정의하면 여러 사이트에 동일 적용 가능 |
| PI System 통합 | 업계 표준 데이터 히스토리안과 네이티브 연계 |
| 40년+ 경험 | 산업 소프트웨어 분야 최장수 기업 중 하나 |
| 하이브리드 클라우드 | 2024년 에지+클라우드 하이브리드 MES 출시 |

**적합 대상**: 석유/가스, 화학, 광업, 식음료 (연속공정 중심)
**주의점**: 이산 제조 기능 한계, 클라우드 전환 상대적 늦음

---

### 2.8 Oracle Manufacturing Cloud

**한 줄 요약**: 처음부터 클라우드로 설계된 SaaS 전용 MES

| 특장점 | 상세 설명 |
|--------|---------|
| 클라우드 퍼스트 | SaaS 전용 설계, 인프라 관리 부담 제로 |
| Oracle ERP 통합 | Fusion ERP/SCM과 네이티브 연계로 엔드투엔드 관리 |
| 규제 산업 지원 | 제약, 의료기기, 식음료 규정 준수 기능 내장 |
| 이산+프로세스 | 두 가지 제조 유형 모두 지원 |

**적합 대상**: Oracle ERP 사용 기업, 클라우드 올인 전략 기업
**주의점**: 커스터마이징 옵션 제한, 벤더 락인 우려, 높은 학습 곡선

### 2.9 Aegis FactoryLogix

**한 줄 요약**: 전자 조립(PCBA/SMT) 분야의 전문 MES

| 특장점 | 상세 설명 |
|--------|---------|
| 전자 조립 특화 | PCBA/SMT 공정에 최적화된 기능 세트 |
| IIoT 데이터 백본 | IIoT 기반 데이터 수집 프레임워크 내장 |
| 모듈형 라이선스 | 필요한 기능만 선택 → 합리적 가격 |
| 강력한 추적성 | 리비전 관리 및 양방향 추적성 우수 |

**적합 대상**: 전자/PCBA 제조, 항공우주 전자, 의료기기
**주의점**: 대규모 사용 시 성능 이슈, 자동 저장 미흡

### 2.10 Critical Manufacturing MES

**한 줄 요약**: 반도체/전자 규제 산업에 특화된 현대적 아키텍처의 MES

| 특장점 | 상세 설명 |
|--------|---------|
| 반도체/전자 특화 | Fab 환경에 최적화된 기능 (웨이퍼 추적, 레시피 관리 등) |
| 현대적 아키텍처 | .NET + Angular/TypeScript, Docker/Kubernetes, 3-Tier |
| 로우코드 확장 | 비개발자도 워크플로 커스터마이징 가능 |
| 최고 사용자 평점 | Gartner 4.5/5.0, 93% 추천률 |
| AR/디지털 트윈 | 증강현실 및 디지털 트윈 기능 내장 |

**적합 대상**: 반도체, 전자(SMT), 의료기기, 항공우주
**주의점**: 가파른 학습 곡선, 높은 도입 가격, 하드웨어 요구 높음
